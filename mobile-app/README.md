# Aplicación Móvil — React Native (Expo)

## Instalación y ejecución

1. Instala dependencias:
   ```bash
   npm install
   ```
2. Ejecuta en modo desarrollo:
   ```bash
   npx expo start
   ```
3. Escanea el QR con Expo Go (Android/iOS) o usa emulador.

## Librerías recomendadas
- `react-native-paper` (UI ejecutiva)
- `react-navigation` (navegación)
- `react-native-chart-kit` (gráficos)
- `axios` (peticiones HTTP)

## Publicación
- Apple App Store
- Google Play Store

## Acceso global
- Publica el backend en la nube (Azure/AWS/Render) para acceso desde cualquier ubicación.
