# Solución Empresarial Móvil Multiplataforma

## Estructura de Carpetas

- `/mobile-app` — Aplicación móvil React Native (Expo)
- `/backend` — API REST Node.js (Express) + conexión segura a SQL Server

## Características principales
- Login seguro con JWT
- UI profesional y ejecutiva
- Backend desacoplado, sin exponer datos de SQL Server
- Ejemplo de conexión segura y buenas prácticas

## Instalación y despliegue
Ver instrucciones detalladas en las carpetas `mobile-app` y `backend`.
